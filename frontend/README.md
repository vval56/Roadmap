# RoadMap React SPA

## Start
```bash
cd frontend
npm install
npm run dev
```

Frontend URL: `http://localhost:5173`

## API URL
By default the app uses `http://localhost:8080/api`.

You can override it with env variable:

```bash
cp .env.example .env
```

Then set:

```env
VITE_API_BASE_URL=http://localhost:8080/api
```
